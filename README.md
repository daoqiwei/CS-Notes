## :memo: 后记

### 排版

笔记内容按照
[中文文案排版指北](https://github.com/sparanoid/chinese-copywriting-guidelines/blob/master/README.zh-CN.md)
进行排版，以保证内容的可读性。

原项目不使用 `![]()` 的方式引用图片，而是使用 `<img>` 标签。
这样一方面能够控制图片显示尺寸，另一方面可以通过：

```html
<div align="center">
    <img src="图片地址">
</div>
```

实现图片居中显示。

在线排版工具：

[Text-Typesetting](https://github.com/CyC2018/Text-Typesetting)
